# Industry Insights

Knowledge extracted from content sources (podcasts, YouTube, industry news)
by the Content Watcher agent.

## How This Works

1. Content Watcher monitors sources in `registry/content-sources.yaml`
2. New episodes/videos are transcribed and analyzed
3. Relevant insights are filed as GitHub issues
4. Team-approved insights are documented here

## Files

- `pricing-models.md` — Beverage industry pricing patterns
- `competitive-landscape.md` — Competitor analysis and market positioning
- `distribution-trends.md` — Distribution channel evolution and trends
